Release 1.0-R1 (Feb 24, 2021)

Features

  • Remote API now allows streaming of collected 1D data sets to clients
  • Remote API added new motion tasks:
    • Reset Encoder
    • Get Axis Position
    • Set Axis Position
    • Axis Go To
    • Axis Jog
    • Axis Run
    • Stop Axis
    • Stop All Axes

Changes

  • Scanner Landmarks page UI has been simplified
  • Axis Homing page UI has been simplified

Fixes

  • Changing channels on Data Collection page with automatic display building would no change the current AScan.
  • Using the Import Setup option would change the name of the current open setup file.
  • Transducer Part and Serial Numbers not presnt in the Data Header
  • Incorrect difference reported between Primary and Secondary cursors on 8bit data

Drivers

  • FocusPX - OpenView Version 1.1.70.1 or greater (Download here, Password: OlympusOpenView)
  • Galil Motion Controller - gclib (Download here)

Release 1.0-RC8 (Dec 15, 2020)

Features

  • Encoder calibration for encoders can be changed on Data Collection page (only available when Load from Setup option is enabled on the specific axis in Motors & Encoders)

Fixes

  • Fixed wrong checksum being generated for data generated in JSON format
  • Focus Length units when creating Focal Laws follows application units now
  • Beam selector on sweeps would not show current beam number
  • Beam selector would not work on Calibration pages

Drivers

  • FocusPX - OpenView Version 1.1.67.3 or greater (Download here, Password: OlympusOpenView)
  • Galil Motion Controller - gclib (Download here)

Release 1.0-RC7 (Oct 10, 2020)

Features

  • New bug reporting tool in Help > Bug Report to generate a bug report package that can be emailed to Utex.
  • IW Scan download page can be reached directly using Help > Check For Updates
  • Remote controllers can now request current position of cursor on scanned data

Changes

  • "Include In Setup File" setting on motion axes has been replaced with "Load From Setup File". The former preference will not be converted, user must re-select the appropriate setting. It is also recommended to resave any setup file in use.*
  • Cursors on cscans now display their position with 3 decimal places of precision.

Fixes

  • Fixed OpenView synchro echo mode offset from zero in some cases
  • OpenView devices have better connection errors
  • Fixed beam number selector disappearing sometimes
  • Fixed encoder preset units always being in mm for 1D and 2D recorders

Drivers

  • FocusPX - OpenView Version 1.1.66.10 or greater (Download here, Password: OlympusOpenView)
  • Galil Motion Controller - gclib (Download here)

Release 1.0-RC6 (Sept 11, 2020)

Features

  • Added the ability to reorder ProbeSets and Channels.*
  • New unified palette editor for all IW Suite applications.
  • Current setup and data file names are shown in the title bar instead of only setup file name.
  • Added a display preference to launch the application with navigation bar collapsed or expanded.
  • Added ability to print the entire histogram window to PDF.

Fixes

  • Tooltips on gates are now converted to inches
  • Tooltips on tvg points are now converted to inches
  • Expanding list of display sources now display a scrollbar when list is longer than window size
  • Removed exit popup when right clicking anywhere in the UI
  • Better handling of units when loading older machine configurations
  • Scan freezing when the palette is changed on a CScan.
  • Fixed annotations not being selectable on some display plots.
  • Removed being able to drop annotations in histogram window.

Drivers

  • FocusPX - OpenView Version 1.1.64.3 or greater (Download here, Password: OlympusOpenView)
  • Galil Motion Controller - gclib (Download here)

*Note: Features or Fixes marked in orange may affect operation of the application at a fundamental level.
